# Contributing

*Claude Code: Principles* is a curated, authored essay. README.md is the document.

**Corrections and small fixes:** open a pull request against README.md. Typos, broken links, clarifications: a precise diff is the fastest path.

**Disagreements and new principles:** open an issue first. Whether a principle holds, what's missing, what contradicts something else: these get worked through in discussion. The essay is curated, not crowd-edited, so substantive changes start as conversations and the author folds them into the text.
